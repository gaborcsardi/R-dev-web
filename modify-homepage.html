HOWTO Modify R's Homepage

Most files at R's homepage at www.R-project.org can be directly modified by all R core mebers using the usual CVS repository on cvr.R-project.org. Simply do a 
 cvs -d cvs.r-project.org:/home/rdevel/CVS-ARCHIVE checkout R-project-web
to get the pages. All changes in the repository are synced to www.R-project.org in Vienna daily at approx. 6:30am CET (=GMT+1).

The only exception is the subtree www.R-project.org/nocvs which contains some automatically generated (and/or huge) parts like the HTMLified mail archives. It makes no sense to put these pages into CVS.
