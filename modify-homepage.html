HOWTO Modify R's Homepage

Most files at R's homepage at www.R-project.org can be directly modified by all R core mebers using the usual subversion repository on svn.R-project.org. Simply do a 
 svn co https://svn.r-project.org/R-project-web/trunk  R-project-web
to get the pages. All changes in the repository are synced to www.R-project.org in Vienna daily at approx. 6:30am CET (=GMT+1).

The only exception is the subtree www.R-project.org/nocvs which contains some automatically generated (and/or huge) parts like the HTMLified mail archives. It makes no sense to put these pages into version control.
