BDR's TODO list

This page contains my plans for enhancements to R. Remember R comes with absolutely no warranty, and that applies to this site and this page in particular.

Brian Ripley ripley@stats.ox.ac.uk

  • A bitmap graphics device, covering at least png, bmp and jpeg on UNIX, as we have on Windows already. This will at least partially replace dev2bitmap. [DONE for 1.1.o]
  • A cleaner approach to `files' and URLs. Possibly something along the lines of S4's connections.
  • The ability to write WMF on Unix (but it looks as if we know too little to achieve this).
  • Port the GraphApp interface used on Windows to Unix, via the Motif version of GraphApp.
  • Something better than gam for additive models. Perhaps along the lines of BRUTO (in package mda).
  • Replace arima0 by better internal code and a better interface.
  • More time-series models, e.g. Harvey's structural models.
