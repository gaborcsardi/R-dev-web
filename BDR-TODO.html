BDR's TODO list

This page contains my plans for enhancements to R. Remember R comes with absolutely no warranty, and that applies to this site and this page in particular.

Brian Ripley ripley@stats.ox.ac.uk

  • Add internal support for XML, based on libxml. [Intended for 1.3.0.]
  • More functionality for connections: sockets, URLs, ...
    Planned for 1.3.0:
    • User-specified line endings when writing text files.
    • URI connections, for types at least file:, http: and ftp:.
    • Implement a stack of sink connections.
    • Ability to sink the error output stream.
  • Port the GraphApp interface used on Windows to Unix, via the Motif version of GraphApp. [This looks much harder than expected due to lack of portability of GraphApp.]
  • Something better than gam for additive models. Perhaps along the lines of BRUTO (in package mda). [Package mgcv has since gone a long way in this direction.]
  • Replace arima0 by better internal code and a better interface.
  • More time-series models, e.g. Harvey's structural models.
  • Something on GLMMs.
Already in un-released versions of R:
  • A native PDF graphics driver. [Done for 1.3.0.]
  • Allow user-specified encodings for postscript output. [Done for 1.3.0.]
