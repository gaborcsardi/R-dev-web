RSS Feeds of R Development News

Daily changes to the NEWS file (for all platforms) and the CHANGES file (for Windows) are now being posted online. We are using R code to calculate diffs, the google-diff-match-patch library to code them in pretty HTML, and Blosxom to display them, so they may be displayed in HTML format or used as RSS feeds to news aggregators such as Feedly.
